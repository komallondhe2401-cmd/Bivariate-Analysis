# 📊 Bivariate Analysis on Multiple Datasets

## 📌 Project Overview

This project demonstrates **Bivariate Analysis** on multiple datasets including `train.csv`, `flights.csv`, `iris.csv`, and `tips.csv`. The objective is to analyze relationships between two variables using various visualization techniques.

---

## 🎯 Objectives

* To study relationships between two variables
* To identify trends, correlations, and patterns
* To compare different features across datasets

---

## 🛠️ Tools & Libraries Used

* Python 🐍
* Pandas
* Matplotlib
* Seaborn

---

## 📂 Datasets Used

* `train.csv` (Kaggle dataset)
* `flights.csv` (Seaborn built-in dataset)
* `iris.csv` (Seaborn built-in dataset)
* `tips.csv` (Seaborn built-in dataset)

---

## 📊 Bivariate Analysis Techniques Used

### 1️⃣ Scatter Plot

* Shows relationship between two numerical variables
* Helps identify correlation and trends

### 2️⃣ Bar Plot

* Compares numerical values across categories
* Useful for aggregated analysis

### 3️⃣ Box Plot

* Compares distribution of a numerical variable across categories
* Helps detect outliers

### 4️⃣ Distplot

* Shows distribution of a variable with density curve
  *(Note: Deprecated, replaced by `histplot`)*

### 5️⃣ Line Plot

* Shows trends over time or ordered data

### 6️⃣ Heatmap

* Visualizes correlation between variables
* Uses color intensity to represent values

### 7️⃣ Clustermap

* Groups similar data points using hierarchical clustering
* Useful for pattern recognition

### 8️⃣ Pairplot

* Shows pairwise relationships between multiple variables
* Provides a complete overview of dataset relationships
